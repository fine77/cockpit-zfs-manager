# Alpha Release Notes

Release: `v0.1.0-alpha.1`
Date: 2026-03-08

## Included

- ZFS UI stabilization and Cockpit theme alignment
- NFS module with managed + active state handling
- SMB module with managed + active state handling
- External entries rendered read-only in module tables

## Known limitations

- SMART feature availability depends on host capabilities
- Minor visual edge cases may still exist in complex table hover states

## Intended use

- Homelab and controlled pilot environments
- Feedback-driven hardening before beta
